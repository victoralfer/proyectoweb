Etiqueta FORM

La etiqueta 'form' se utiliza para crear formularios que permiten al usuario enviar información a un servidor.

Atributo Method
Parámetro "get"

< form method="get" >
</ form >

Explicación: indica como se envían datos al servidor. No es habitual su uso ya que la información queda expuesta a cualquier usuario. Se recomienda sólo para consultas en un buscador.

Atributo Method
Parámetro "post"

< form method="post" >
</ form >

Explicación: indica como se envían datos al servidor. Su uso es el más habitual ya que la información se procesa de una manera más segura a través de los protocolos HTTP o HTTPS (encriptado).

Atributos
Action y Enctype

< form method="post" action="php/formulario.php" enctype="multipart/form-data" >
</ form >

Explicación: estos atributo se utilizan cuando el método elegido para el envío del formulario es POST. El parámetro 'action' lleva consigo la ruta del archivo php con la configuración de envío al servidor.

Etiqueta INPUT

La etiqueta 'input' se utiliza dentro de un formulario para que el usuario ingrese o seleccione datos que luego serán enviados al servidor.

Se personaliza con el atributo 'type', que define el tipo de dato a capturar.

Siempre debe llevar el atributo 'name', que será la clave con la que se enviará la información.

Atributo Name

< input name="edad" >

Explicación: es el nombre del campo, el cual se utiliza como clave para enviar la información al servidor.

Atributo Value

< input name="edad" type="number" value="27" >

Explicación: se utiliza para introducir un valor inicial a un campo.

INPUT | Type

Parámetro: text

< input name="nombre" type="text" >

Explicación: tipo de campo de texto básico.

Parámetro: number

< input name="edad" type="number" >

Explicación: tipo de campo numérico.

Parámetro: password

< input name="pass" type="password" >

Explicación: tipo de campo para contraseñas. Se muestra con asteriscos.

Parámetro: submit

< input name="envio" type="submit" value="Enviar" >

Explicación: botón para enviar el formulario. No es necesario complementarlo con el atributo 'name'. El atributo 'value' define el texto que aparece en el botón.

Parámetro: reset

< input name="borrar" type="reset" value="Borrar" >

Explicación: botón para resetear el formulario. No es necesario complementarlo con el atributo 'name'. El atributo 'value' define el texto que aparece en el botón.

Parámetro: button

< input name="boton" type="button" value="Descarga" >

Explicación: se utiliza para crear un botón genérico. Su función se define con JavaScript. No es necesario complementarlo con el atributo 'name'. El atributo 'value' define el texto que aparece en el botón.

Type: Especiales

Parámetro: color

< input name="color" type="color" value="#FFF000" >

Explicación: se utiliza para mostrar un selector de colores que permite la elección de un color al usuario.

El atributo value selecciona un color predeterminado que se enviará al servidor en el caso de que el usuario no seleccione ninguno.

Parámetro: file

< input name="adjuntar" type="file" >

Explicación: se utiliza para permitir a los usuarios adjuntar archivos.

Parámetro: tel

< input name="telefono" type="tel" >

Explicación: es un campo específico para el número de teléfono.

Parámetro: email

< input name="email" type="email" >

Explicación: campo para correos electrónicos, que incluso valida su formato.

Parámetro: url

< input name="web" type="url" >

Explicación: campo para introducir direcciones web. A la hora de incluir las direcciones se necesita poner los protocolos HTTP o HTTPS.

Parámetro: date

< input name="fecha" type="date" >

Explicación: campo para introducir una fecha a través de un calendario emergente.

Parámetro: time

< input name="hora" type="time" >

Explicación: campo para introducir la hora a través de un menú emergente.

Parámetro: datetime-local

< input name="fecha-hora" type="datetime-local" >

Explicación: campo para introducir tanto la fecha como la hora.

Parámetro: month

< input name="mes" type="month" >

Explicación: campo para introducir el mes, acompañado de un año a seleccionar.

Type: Selecciones

Parámetro: range

< input name="volumen" type="range" min="0" max="100" step="5" value="50" >

Explicación: es un parámetro que sirve para crear un deslizador (slider) que permite al usuario seleccionar un valor numérico dentro de un rango definido.

Se puede complementar con otros atributos para aportarle otras características.

  • min: valor mínimo del rango
  • max: valor máximo del rango
  • value: valor inicial el rango
  • step: incremento al mover el deslizador

Parámetro: checkbox

< input name="check" type="checkbox" value="opcion1" >

Explicación: Se utiliza para crear una casilla de verificación que permite al usuario seleccionar o deseleccionar una o varias opciones dentro de un formulario.

Si hay varias opciones no es necesario utilizar el mismo 'name', pero es una buena práctica hacerlo para enviar al servidor los datos seleccionados en un solo array.

El atributo 'value' asigna un valor diferente a cada opción.

Parámetro: radio

< input name="opciones" type="radio" value="opcion1" >

< input name="opciones" type="radio" value="opcion2" >

Explicación: Se utiliza para crear botones circulares con diferentes opciones para el usuario. Sólamente se puede seleccionar una alternativa.

Requiere que todas las opciones compartan el mismo 'name'. Cada opción tiene un 'value' diferente.

INPUT | Otros

Atributo placeholder

< input name="nombre" type="text" placeholder="Víctor" >

Explicación: se utiliza para mostrar una ayuda visual para el relleno de los campos.

Atributo maxlength

< input name="nombre" type="text" maxlength="255" >

Explicación: se utiliza para fijar un número máximo de caracteres a rellenar en un campo.

Atributo required

< input name="nombre" type="text" required >

Explicación: se utiliza para indicar que un campo es de relleno obligatorio.

Atributo disabled

< input name="nombre" type="text" disabled >

Explicación: se utiliza para deshabilitar un campo. El campo se mostrara a los usuarios, pero no permitirá su modificación.

Atributo hidden

< input name="nombre" type="text" hidden >

Explicación: se utiliza para ocultar un campo. El campo no se mostrará a los usuarios.

Atributo readonly

< input name="nombre" type="text" readonly >

Explicación: es un campo de sólo lectura que no permite modificación por parte del usuario.

Atributo tabindex

< input name="nombre" type="text" tabindex="1" >

Explicación: se incorpora para establecer una prioridad en el relleno de campos cuando se utiliza el tabulador para saltar entre los mismos. Facilita la navegación al usuario a la hora de completar aquellos campos que bien son requeridos o tienen mayor importancia.

Atributo step

< input name="nombre" type="number" step="0.01" >

Explicación: se utiliza para añadir decimales cuando en el atributo type su parámetro es 'number'.

La posición del '1' después del punto, muestra el número de decimales que se aplicarán.

Atributo checked

< input name="nombre" type="checkbox" checked >

Explicación: se utiliza para marcar una de las opciones en los parámetros 'radio' o 'checkbox' del atributo type.

Etiqueta SELECT

Se utiliza dentro de un formulario para crear un menú desplegable que permite al usuario seleccionar una o varias opciones.

Es necesario complementarla con la etiqueta 'option', la cual introducirá dentro de 'select' las diferentes opciones de selección.

Atributo Name

< select name="fruta" >

< option value="manzana" > Manzana </ option >

< option value="naranja" > Naranja </ option >

</ select >

Explicación: se utiliza para nombrar el campo que se enviará al servidor junto con los valores seleccionados por el usuario.

Atributo Size

< select name="profesiones" size="2" >

< option value="informatico" > Informático </ option >

< option value="mecanico" > Mecánico </ option >

< option value="taxista" > Taxista </ option >

</ select >

Explicación: se utiliza para desplegar en pantalla el número de opciones que se indiquen en su configuración.

Atributo Multiple

< select name="asignatura" multiple >

< option value="matematicas" > Matemáticas </ option >

< option value="ingles" > Inglés </ option >

</ select >

Explicación: permite seleccionar varias opciones de lista desplegable.

Etiqueta OPTION

Se utiliza dentro de la etiqueta 'Select' para mostrar diferentes opciones en la lista desplegable.

Atributo Value

< select name="animales" >

< option value="perro" > Perro </ option >

< option value="gato" > Gato </ option >

</ select >

Explicación: permite asignar valores a las diferentes opciones de la lista desplegable.

Atributo Selected

< select name="comida" >

< option value="macarrones" > Macarrones </ option >

< option value="paella" selected > Paella </ option >

</ select >

Explicación: se utiliza para seleccionar un valor predefinido de la lista desplegable.

Etiqueta TEXTAREA

Se utiliza para crear un campo de texto multilínea, es decir, una caja en la que el usuario puede escribir varias líneas de texto, como si fuera un pequeño editor.

Atributo Name

< textarea name="observaciones" >
</ textarea >

Explicación: se utiliza para nombrar el campo de texto que se enviará al servidor junto con el texto escrito por el usuario.

Otros Atributos

< textarea name="observaciones" placeholder="Escribe aquí tu comentario" maxlength="255" tabindex="25" required >
</ textarea >

Explicación: puede utilizar atributos como 'placeholder', 'maxlength', 'tabindex' o 'required'.

Etiqueta LABEL

Se utiliza para etiquetar elementos de formulario, como campos de entrada 'input', menús desplegables 'select' o áreas de texto 'textarea'.

El parámetro del atributo 'for' debe coincidir con el parámetro del atributo 'id' de las etiquetas 'input', 'select' o 'textarea'.

Su principal función es asociar un texto descriptivo a un elemento del formulario, mejorando así la accesibilidad y la experiencia del usuario.

< label for="nombre-apellidos" > Nombre y Apellidos:</ label >

< input id="nombre-apellidos" name="nombre-apellidos" type="text" >

Etiqueta FIELDSET

Se utiliza para crear una capa especial dentro de un formulario, que permite incluir un titulo en el interior de un borde si se incorpora antes de su cierre la etiqueta 'legend'.

< fieldset >

< legend >Datos Personales</ legend >

</ fieldset >

Pseudo-Clases

Se pueden utilizar en las etiquetas 'input', 'select' y 'textarea'.

Valid

input:valid { color: white; background-color: blue; }

Explicación: se utiliza para aplicar propiedades a campos válidos.

Invalid

input:invalid { color: white; background-color: blue; }

Explicación: se utiliza para aplicar propiedades a campos inválidos.

Focus

input:focus { color: white; background-color: blue; }

Explicación: se utiliza para aplicar propiedades a campos pulsados.

Required

input:required { color: white; background-color: blue; }

Explicación: se utiliza para aplicar propiedades a campos requeridos.
