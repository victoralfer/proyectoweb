Introducción

Implementación

jQuery es una librería de JavaScript rápida, ligera y muy utilizada para simplificar la gestión de eventos y animaciones.

Para poder utilizar jQuery en una página web, es necesario incluir la librería mediante una de las siguientes opciones:

  • Descargar el archivo y ponerlo en el servidor.
  • Acceder al servidor externo que contiene la librería (Opción Recomendada).

Las librerías son archivos en formato .js, que se ejecutan en el Cliente.

Etiqueta Script

  • Enlaza con un archivo de código JavaScript.
  • Los enlaces a los archivos de las librerías .js se escriben en el <head> </head>, debajo del link que contiene el CSS.
  • La ruta del archivo con la librería .js se coloca en el atributo src.

<script src="https://code.jquery.com/jquery-3.7.1.js"> </script>

Colocación HEAD

  1. Link con la ruta de la Hoja de Estilos (CSS).
  2. Script con el enlace a la Librería jQuery.
  3. Script con la ruta del archivo JavaScript.

Captura de Objetos

¿Cómo se capturan objetos?

Los objetos capturados para ejecutar funciones en ellos mismos se cargan en el archivo JS que se está utilizando en la página.

  • Capturador y Cargador de Objetos window.onload.
  • Capturador y Cargador de jQuery.
  • Funciones del archivo JS.

Sintaxis jQuery

$(".capas h4").show();

Se utiliza el símbolo $ que se completa entre parentesis con la 'id', 'clase' o 'etiqueta del objeto' que se desea capturar. Tras la captura se ejecutan las funciones deseadas.

Capturador window.onload

window.onload = function() {

$("#principal").hide();

}

Capturador jQuery

$(document).ready (function() {

$("#principal").hide();

});

Captura en una Función

function fun_slide() {

$("h4").slideUp(3000);

}

Captura de Valores

Capturar Valor de un Atributo

  • El atributo tiene que estar en la etiqueta HTML.
  • Uso habitual: coger/modificar datos.

$("#capa").attr("width");

Capturar Valor de un INPUT

$("input").val();

Eventos más usados

  • Un click: click
  • Doble-Click: dblclick
  • Interruptor: toggle
  • Cursor entra en el objeto: mouseover
  • Cursor sale del objeto: mouseout
  • Ejecución de código una vez cargado el contenido de la página: ready
  • Asignación de eventos a elementos HTML: on

Ejecución Elemento: this

Para ejecutar una función en un elemento concreto esta debe cargarse bien en el window.onload o en el capturador jQuery.

$("p").click (function() {

$(this).css({"color": "blue"});

});

  • Tipo de Objetos a Capturar: "p"
  • Evento que ejecuta una función: .click
  • Elemento concreto en el que se quiere ejecutar la función: this
  • Función a ejecutar: .css({"color": "blue"})

Ocultar: hide()

Esta función se utiliza para ocultar un objeto.

$("#principal").hide();

Mostrar: show()

Esta función se utiliza para mostrar un objeto.

$("#principal").show();

Aparición Gradual: fadeIn()

Esta función se utiliza para hacer aparecer de manera gradual un objeto.

$("#principal").hide().fadeIn(3000);

  • Para su funcionamiento es necesario ocultar primero el objeto para que se genere el efecto de aparición.
  • Se establece dentro de los paréntesis el tiempo de duración del efecto en milisegundos (ms).

Desaparición Gradual: fadeOut()

Esta función se utiliza para hacer desaparecer de manera gradual un objeto.

$("#principal").fadeOut(3000);

Se establece dentro de los paréntesis el tiempo de duración del efecto en milisegundos (ms).

Inicio retardado: delay()

Esta función se utiliza combinada con otra función para demorar su inicio durante un tiempo determinado en milisegundos (ms).

$("#principal").delay(5000).fadeOut(3000);

Añadir texto al principio de un elemento: prepend()

Esta función permite añadir tanto texto plano como con estilos dentro de un elemento, ubicando el texto al principio del elemento o elementos.

Ejemplo con Texto Plano

$("#principal").prepend("Texto añadido al principio del elemento");

En este ejemplo se añadiría dentro de la capa con id 'principal' un texto plano (sin estilo) que se ubicaría al inicio de la misma.

Ejemplo de Texto con Estilo

$("#principal").prepend("<h1> Texto añadido con estilo al principio del elemento </h1>");

En este ejemplo se añadiría dentro de la capa con id 'principal' un texto con estilo que se ubicaría al inicio de la misma.

Añadir texto al final de un elemento: append()

Esta función permite añadir tanto texto plano como con estilos dentro de un elemento, ubicando el texto al final del elemento o elementos.

Ejemplo con Texto Plano

$("#principal").append("Texto añadido al final del elemento");

En este ejemplo se añadiría dentro de la capa con id 'principal' un texto plano (sin estilo) que se ubicaría al final de la misma.

Ejemplo de Texto con Estilo

$("#principal").append("<h1 class="titulos"> Texto añadido con estilo al final del elemento </h1>");

En este ejemplo se añadiría dentro de la capa con id 'principal' un texto con estilo que se ubicaría al final de la misma.

Añadir contenido antes de un elemento: before()

Esta función permite añadir contenido antes de un elemento.

Ejemplo

$("#principal").before("<div id='antes'> Capa añadida antes de la indicada </div>");

En este ejemplo se añadiría antes de la capa con id 'principal' otra cuya id sería 'antes' y que llevaría como contenido el texto "Capa añadida antes de la indicada".

Añadir contenido después de un elemento: after()

Esta función permite añadir contenido después de un elemento.

Ejemplo

$("#principal").after("<div id='despues'> Capa añadida después de la indicada </div>");

En este ejemplo se añadiría después de la capa con id 'principal' otra cuya id sería 'despues' y que llevaría como contenido el texto "Capa añadida después de la indicada".

Sustituir contenido por texto plano: text()

Esta función permite sustituir contenido de un elemento por texto plano.

Ejemplo

$("p").click (function() {

$(this).text("Este texto sustituye el contenido del elemento indicado");

});

En este ejemplo se sustituye el contenido de la etiqueta 'p' que es seleccionada con un click, por un texto plano (sin estilo).

Sustituir contenido por texto con estilo: html()

Esta función permite sustituir contenido de un elemento por texto con estilo.

Ejemplo

$("p").click (function() {

$(this).html("<span class='parrafo-principal'> Este texto sustituye el contenido del elemento indicado </span>");

});

En este ejemplo se sustituye el contenido de la etiqueta 'p' que es seleccionada con un click, por un texto con estilo.

Añadir una clase a un elemento: addClass()

Esta función permite añadir una clase a un elemento.

Ejemplo

$("p").click (function() {

$(this).addClass("negritas");

});

En este ejemplo se añade una clase a un elemento concreto de un párrafo seleccionado con un click.

Elimina una clase a un elemento: removeClass()

Esta función permite eliminar una clase a un elemento.

Ejemplo

$("h1").click (function() {

$(this).removeClass("cursivas");

});

En este ejemplo se elimina una clase a un elemento concreto de un título h1 seleccionado con un click.

Eliminar contenido de un elemento: empty()

Esta función permite eliminar el contenido interno de un elemento.

Ejemplo

$("#capa").empty();

En este ejemplo se elimina el contenido interno del elemento con la id 'capa'.

Eliminar un elemento: remove()

Esta función permite eliminar un elemento.

  • Eliminación de Elementos 'Tags' o 'Id': desde cualquier cargador de objetos.
  • Eliminación de Elementos Concretos de una Clase: desde el cargador de jQuery ($(document).ready()).

Ejemplo para eliminar un elemento que sea una etiqueta o una id

$("#explicacion").remove();

En este ejemplo se elimina el elemento con la id 'explicacion'. Estos elementos se pueden eliminar tanto desde una función, como desde el cargador de jQuery ($(document).ready()) e incluso desde el cargador nativo de JavaScript (window.onload).

Ejemplo para eliminar un elemento concreto de una clase

$(document).ready (function() {

$(document).on ("click", ".principales", function() {

$(this).remove()

});

});

Para eliminar un elemento concreto de una clase, esta se debe realizar cargando dicha función desde el cargador de jQuery ($(document).ready()).

Efecto persiana subiendo: slideUp()

Esta función permite añadir un efecto que simula una persiana subiendo.

Ejemplo

$("#contenido").slideUp(4000);

Se establece dentro de los paréntesis el tiempo de duración del efecto en milisegundos (ms).

Efecto persiana bajando: slideDown()

Esta función permite añadir un efecto que simula una persiana bajando.

Ejemplo

$("#contenido").slideDown(3500);

Se establece dentro de los paréntesis el tiempo de duración del efecto en milisegundos (ms).

Efecto persiana con interruptor: slideToggle()

Esta función permite añadir un efecto que simula una persiana que sube o baja tras activarse un evento.

Ejemplo

$("#contenido").slideToggle(2500);

Se establece dentro de los paréntesis el tiempo de duración del efecto en milisegundos (ms).

Añadir propiedades CSS: css()

Esta función se utiliza para añadir propiedades de CSS a un elemento.

Añadir una propiedad

$("#capa").css({"color": "green"});

Añadir varias propiedades

$("#capa").css({"color": "green", "border": "1px solid black"});

Animación de un elemento: animate()

Esta función permite generar un efecto utilizando un reducido número de propiedades de CSS. La función 'animate' se debe implementar a través del cargador de jQuery.

Ejemplo

$(document).ready (function() {

$("#visor img").mouseover (function() {

$(this).css({"border": "0px solid red"}).animate({

opacity: 0.5,

width: "100px",

}, 700);

});

});

});

En este ejemplo se establece una animación que activa determinadas características de CSS, cuando el cursor del ratón pasa por encima de una imagen que pertenece a la id 'visor'. La duración de la animación es de 700 milisegundos.

Enlace de ancla: animate({scrollTop: 0})

La función animate junto con la propiedad scrollTop permite realizar una función de ancla al inicio de la página, generando un efecto de subida progresivo.

Ejemplo

$(document).ready (function() {

$("#ancla").on ("click", function() {

$(html).animate({

scrollTop: 0

}, 600);

});

});

});

En este ejemplo se establece una animación que realiza un ancla al incio de la página ('html'). La duración de la animacion es de 600 milisegundos.
