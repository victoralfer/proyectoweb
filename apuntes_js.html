Introducción

Formato de los Archivos: .js

Los archivos se ejecutan en el Cliente.

Etiqueta Script

  • Enlaza con un archivo de código JavaScript.
  • Permite también añadir código de manera directa (opción poco utilizada).
  • Los enlaces a los archivos .js se escriben en el <head> </head>, debajo del link que contiene el CSS.
  • La ruta del archivo .js se coloca en el atributo src.

<script type="text/javascript" src="js/javascript.js"> </script>

Variables

Tipos de Datos:

  • Numéricos (sin comillas): nota = 2;
  • String (con comillas): nombre = "Victor";
  • Booleano (True/False): activo = true;
  • Objetos: Captura de etiquetas del HTML.

Declaración de Variables

  • var
  • let (Opción más usada. No genera conflictos entre variables locales y globales): let edad;

Iniciación de Variables

Ejemplo: edad = 2;

Declaración + Iniciación

  • Numéricos: let nota = 8;
  • Numéricos con Decimales: let precio = 15.2;
  • String: let comentario = "Bien hecho";
  • Booleano: let activo = false;

Constantes: const

Usos

Se utilizan constantes cuando se necesitan valores que no sufrirán modificaciones.

Declaración e Iniciación

Se declaran e inician en un solo paso.

Ejemplo

const dilatacion = 0.3;

Alertas: alert()

Tipos de Muestras

  • Variables: alert (suma);
  • Textos: alert ("Hola Mundo");
  • Concatenado (Texto + Variable): alert ("La edad de: " + nombre + " es:" + edad);

Salto de Línea: \n

Sólo funcionan con las alertas.

Ejemplo

alert ("La edad de " + nombre + " es:" + "\n" + edad);

Operaciones Básicas

Suma

Se realiza con el operador +, utilizando constantes o variables de 'tipo numérico'.

let edad1 = 5;

let edad2 = 10;

let suma = edad1 + edad2;

Resta

Se realiza con el operador -, utilizando constantes o variables de 'tipo numérico'.

let nota1 = 10;

let nota2 = 4;

let resta = nota1 - nota2;

Multiplicación

Se realiza con el operador *, utilizando constantes o variables de 'tipo numérico'.

let arboles = 15;

let multiplicacion = arboles * 2;

División

Se realiza con el operador /, utilizando constantes o variables de 'tipo numérico'.

let tortillas = 10;

let division = tortillas / 5;

Otras Operaciones

Resto de una División

Se realiza con el operador %, utilizando constantes o variables de 'tipo numérico'.

let sillas = 12;

let mesas = 2;

let resto_division = sillas % mesas;

Potencias

Se realiza con el operador **, utilizando constantes y variables de 'tipo numérico'.

let pantallas = 2;

let cuadrado = pantallas ** 2;

let cubo = pantallas ** 3;

Incrementos

Opción 1: a++

Se realiza primero la operación y después el incremento de la variable.

let a = 2;

let b = 3;

let suma = a++ + b;

// suma = 5, nuevo valor de a: 3

Opción 2: ++a

Se realiza primero el incremento de la variable y después la operación.

let a = 2;

let b = 3;

let suma = ++a + b;

// suma = 6, nuevo valor de a: 3

Decrementos

Opción 1: a--

Se realiza primero la operación y después el decremento de la variable.

let a = 2;

let b = 3;

let resta = a-- + b;

// resta = -1, nuevo valor de a: 1

Opción 2: --a

Se realiza primero el decremento de la variable y después la operación.

let a = 2;

let b = 3;

let resta = --a + b;

// resta = -2, nuevo valor de a: 1

Asig. Compuesta

Otros Incrementos

Incremento de '2' en '2': i+=2;

Incremento de '3' en '3': i+=3;

Incremento multiplicado por '3': i*=3;

Otros Decrementos

Decremento de '2' en '2': i-=2;

Decremento de '3' en '3': i-=3;

Decremento dividido por '3': i/=3;

Explicación

Se utiliza para comparar valores y determinar si son o no distintos.

  • Valores iguales: 'false'

let total = a != b;

// Resultado: total es 'false'

Explicación

Se utiliza para comparar valores y determinar si son mayores o menores.

  • Valores Ciertos: 'true'
  • Valores Inciertos: 'false'

Ejemplo 'Menor que' <

let a = 10;

let b = 3;

let total = a < b;

// Resultado: total es 'false'

Ejemplo 'Mayor que' >

let a = 10;

let total = a > b;

// Resultado: total es 'true'

let a = 10;

let total = a <= b;

// Resultado: total es 'false'

let a = 10;

let b = 3;

let total = a >= b;

// Resultado: total es 'true'

Estrictamente

Explicación

Se utiliza para comparar valores y determinar si son estrictamente iguales o distintos.

  • Valores Iguales: 'true'
  • Valores Distintos: 'false'

Ejemplo Estrictamente Igual ===

let a = 10;

let b = 10;

let total = a === b;

// Resultado: total es 'true'

Ejemplo Estrictamente Distinto !==

let a = 10;

// Resultado: total es 'true'

Operador AND &&

Explicación

Este operador se utiliza cuando se requiere el cumplimiento de todas las opciones propuestas.

Se recomienda el uso de paréntesis para delimitar las opciones.

Posibilidades:
  • Se cumplen todas las opciones: 'true'
  • No se cumplen todas las opciones: 'false'

Ejemplo

let a = 2;

let b = 2;

let c = 4;

let d = 4;

let total = (a == b) && (c == d);

// Resultado: total es 'true', se cumple tanto la primera como la segunda opción.

Operador OR ||

Explicación

Este operador se utiliza cuando se requiere el cumplimiento de al menos una de las opciones propuestas.

Se recomienda el uso de paréntesis para delimitar las opciones.

Posibilidades:
  • Se cumple al menos una opción: 'true'
  • No se cumple ninguna opción: 'false'

Ejemplo

let a = 2;

let b = 2;

let c = 4;

let d = 5;

let total = (a == b) || (c == d);

// Resultado: total es 'true', se cumple sólo la primera opción.

Valores Boolenaos

Explicación

Los valores booleanos en JavaScript son 'true' y 'false'. Se generan a partir de operaciones lógicas y de comparación y son fundamentales para la toma de decisiones en el código.

VALORES TRUE

  • Numeros Enteros (distintos de 0): 1, 2, 3, ...
  • Caracteres: "a", "J", ...
  • Cadena de Caracteres: "Hola", "¿Cómo estás?", ...

VALORES FALSE

  • 0
  • Cadena de Caracteres Vacía: " ".
  • NaN (No es un número): este hecho se da cuando se están utilizando números y se introduce o compara con algo que no lo es.
  • Null (campo vacío)
  • undefined (sin definir)

Ejemplo

let a = 3;

let b = 5;

let total = !a == b;

/* Resultado: total es 'false'

  1. El valor 'a' al ser un número entero es 'true'. Este valor se convierte en 'false' por el símbolo !.
  2. El valor 'b' al ser un número entero es 'true'.
  3. La comparación 'false' (!a) no es igual a 'true' (b). */

Prompt()

¿Qué es el prompt?

El prompt es una función que permite la aparición de una ventana emergente inicial que se utiliza para mostrar alertas (alert) o introducir valores a una variable.

Los datos que se introducen por el 'prompt' son de tipo 'string'.

Ejemplo de Uso

let edad = prompt("Escribe tu edad: ");

Conversión a Dato Numérico

Para convertir un dato introducido por el 'prompt' en numérico será necesario multiplicar el dato por '1' o utilizar la función Number().

Es indispensable realizar esta conversión cuando se pretenden sumar valores. Si no se realiza dicha conversión el operador + se interpretará como una concatenación.

Ejemplos de Conversión

Multiplicando por 1

let edad = prompt("Escribe tu edad: ") * 1;

Utilizando la función Number()

let edad = Number(prompt("Escribe tu edad: ") );

Indica el tipo de variable si la mostramos a través de una alerta tanto en el prompt como en el HTML.

Ejemplo Variable Numérica

let numero = 1;

alert(typeof(numero));

// Tipo de Dato: 'number'

Ejemplo Variable String

let nombre = "Pepe";

alert(typeof(nombre));

// Tipo de Dato: 'string'

Ejemplo Variable Boolena

let activo = true;

alert(typeof(activo));

// Tipo de Dato: 'boolean'

Math: Número Aleatorio

Explicación Math.random

Calcula y muestra un número aleatorio entre 0 y 1, excluyendo este último.

let aleatorio = Math.random();

alert(aleatorio);

// Resultado: mostraría un resultado entre 0 y 1

Número Aleatorio entre 0 y 9

Se multiplica por 10 el valor aleatorio, redondeándolo hacía abajo.

let aleatorio = Math.floor(Math.random() * 10);

Número Aleatorio entre 1 y 10

Se multiplica por 10 el valor aleatorio, redondeándolo hacía abajo y sumándole 1.

Se recomienda utilizar el redondeo hacía abajo, sumándole posteriormente 1, para que la probabilidad de que salgan todos los números sea más precisa.

let aleatorio = Math.floor(Math.random() * 10) + 1;

Número Aleatorio entre 1 y 6: Dado

Se multiplica por 6 el valor aleatorio, redondeándolo hacía abajo y sumándole 1.

Se recomienda utilizar el redondeo hacía abajo, sumándole posteriormente 1, para que la probabilidad de que salgan todos los números sea más precisa.

let aleatorio = Math.floor(Math.random() * 6) + 1;

Mostrar Caracteres Concretos

.charAt()

Esta función se utiliza para mostrar el caracter de una posición determinada de una variable con valor de tipo 'string'.

La primera posición corresponde con el número '0', la segunda posición con el número '1', la tercera posición con el número '2', y así sucesivamente.

let nombre = "Víctor";

nombre.charAt(2);

alert(nombre);

// Resultado: 'c'. Es el caracter que corresponde con el número 2, especificado entre paréntesis dentro de la función.

.substring()

Muestra los caracteres entre dos posiciones concretas.

  1. El primer número de la función se asocia con la posición inicial de la cadena que se mostrará.
  2. El segundo número de la función se asocia con la posición final de la cadena que se mostrará, no incluyéndose la misma en la muestra.
  3. Si tan sólo se indica un número, se devuelven los caracteres desde esa posición hasta el final de la cadena.

let nombre = "Víctor";

nombre.substring(2, 5);

alert(nombre);

// Resultado: "cto". Muestra los caracteres de las posiciones 2, 3 y 4.

Mostrar Posiciones Concretas

.indexOf()

Muestra en que posición se encuentra el primer caracter igual al indicado.

let nombre = "Víctor";

nombre.indexOf("o");

alert(nombre);

// Resultado: 4. Muestra la posición del primer caracter 'o' de la variable.

.lastIndexOf()

Muestra en que posición se encuentra el último caracter igual al indicado.

let nombre = "Casa";

nombre.lastIndexOf("a");

alert(nombre);

// Resultado: 3. Muestra la posición del último caracter 'a' de la variable.

Array

Es el conjunto de valores que puede tomar una variable.

Ejemplos de Tipos de Arrays

  • Valores 'string': let nombres = ["Víctor", "Juan"];
  • Valores 'numéricos': let notas = [2, 3, 4];
  • Valores 'booleanos': let activo = [true, false];
  • Valores mixtos: let observaciones = ["Bien", 3, 4, true, false];

Posiciones

La primera posición de un array es la número '0', la segunda posición es la número '1', la tercera posición es la número '2', y así sucesivamente.

Mostrar una Posición Concreta

Se coloca el array entre los paréntesis, el cual contendrá la posición deseada entre corchetes.

let notas = [2, 3, 4, 5];

document.write(notas[2]);

// Resultado: 4. Número ubicado en la posición '2'

Mostrar todos los valores

Se coloca simplemente el array entre los paréntesis.

let notas = [2, 3, 4, 5];

document.write(notas);

// Resultados: 2, 3, 4 y 5

Mostrar número de elementos

Se coloca el array entre los paréntesis, seguido de la función .length.

let notas = [2, 3, 4, 5];

document.write(notas.length);

// Resultado: 4. Número total de valores que contiene el array.

Mostrar último elemento

Se coloca el array entre los paréntesis, acompañado entre corchetes del propio array, la función .length y restándole 1.

let notas = [2, 3, 4, 5];

document.write(notas[notas.length - 1]);

// Resultado: 5. Muestra el valor del último elemento.

// Se añadiría el número 9 al principio del array

Condicional Sencillo

No es habitual su uso.

Ejemplo explicativo

Para utilizar un condicional sencillo será necesario el uso de un array, una variable con su valor correspondiente y otra variable cuyo valor será la condición sencilla.

Además, se requiere una alerta que asocie el array con la variable que contiene la condición.

let hola = ["No está", "Sí está"];

let nombre = prompt("Introduce un nombre: ");

let solucion = ((nombre == "Juan") || (nombre == "Pepe"))? 1:0;

alert(hola[solucion]);

Funcionamiento Condicional

  • Lo situado antes de los dos puntos (:) es TRUE.
  • Lo situado después de los dos puntos (:) es FALSE.
Es decir:
  1. Si el nombre es "Juan" o "Pepe" (se cumple la condición), se elige lo situado antes de los dos puntos; es decir, la posición '1' ⇒ "Si está"
  2. Si el nombre no es "Juan" ni "Pepe" (no se cumple la condición), entonces se elige lo situado después de los dos puntos; es decir, la posición '0' del array ⇒ "No está"

Sucesión de Condicionales

Se pueden realizar sucesiones de condicionales para elegir más elementos del array, recordando siempre que:

  • Lo situado antes de los dos puntos (:) se seleccionará si se cumple la condición (TRUE).
  • Lo situado después de los dos puntos (:) se seleccionará si no se cumple la condición (FALSE).

IF / ELSE IF / ELSE

Sintaxis

if (condición) {

código;

}

else if (condición) {

código;

}

else {

código;

}

Observaciones

  • Se puede utilizar sólo 'if'.
  • Se pueden utilizar tantos 'else if' como sean necesarios.
  • Se puede utilizar 'else' combinado simplemente con 'if', sin necesidad de usar 'else if'.
  • Se pueden anidar condicionales.

Ejemplo

if (numero == 0) {

alert("El número es cero");

}

else {

alert("El número es distinto de cero");

}

Explicación del Ejemplo

  • Si (if) la variable 'numero' es 0 se ejecuta la alerta El número es cero.
  • Si no (else), se ejecuta la alerta El número es distinto de cero.

Bucle Indeterminado

- WHILE (mientras) -

Este tipo de bucle permite ejecutar un código mientras se cumpla una condición indeterminada.

Ejemplo Funcionamiento

let numero = prompt("Escribe un número: ");

let aleatorio = Math.floor(Math.random() * 100);

while (numero != aleatorio) {

document.write("No has acertado el número");

}

document.write("Has acertado el número");

Explicación del Ejemplo

  • Mientras (While) la variable 'numero' (que se introduce por el prompt) sea distinta de la variable 'aleatorio', se ejecuta la alerta No has acertado el número.
  • Si la variable 'numero' es igual a la variable 'aleatorio', se ejecuta la alerta Has acertado el número.

Bucle Determinado

- FOR -

Este tipo de bucle permite ejecutar un código mientras se cumpla una condición determinada.

Ejemplo Funcionamiento

let dias = ["L", "M", "X", "J", "V", "S", "D"];

for (i=0; i < dias.length; i++) {

document.write(dias[i]);

}

document.write("Se acabó la semana");

Explicación Condición Bucle For

  • Valor inicial: i=0
  • Condición: i < dias.length
  • Incremento de la variable: i++

En este ejemplo se mostrarían en el HTML todos los valores, comenzando por el que está en la posición '0' y finalizando por el que está en la posición final.

Bucle forEach

Este tipo de bucle permite ejecutar un código una vez por cada elemento de un array, sin necesidad de declarar una variable de control ni una condición explícita.

Ejemplo Funcionamiento

let dias = ["L", "M", "X", "J", "V", "S", "D"];

dias.forEach (i=> {

i.style.padding = "10px 0";

i.style.color = "blue";

i.style.fontSize = "1.5em";

});

Explicación Funcionamiento

  • Se utiliza el array dias, seguido del bucle forEach.
  • Se asigna una variable que permitirá aplicar código a todos los elementos del array: i
  • Se introducen las líneas de código que se desean aplicar a todos los elementos del array: i.codigo

En este ejemplo se modifican varias propiedades de CSS de todos los elementos que componen el array 'dias'.

Creación Funciones

Sintaxis

function nombre_funcion() {

// Código del Programa

}

Ejemplo

function fun_cambiar() {

nombre.forEach (j=> {

j.style.margin = "10px";

j.style.border = "1px solid orange";

j.style.display = "none";

});

}

Eventos en HTML

Se colocan en las etiquetas HTML en las que se quiere ejecutar una función.

Eventos + usados

  • Ejecutar con un click: onclick
  • Ejecutar con doble click: dblclick
  • Ejecutar al pasar el cursor por encima del objeto: onmouseover
  • Ejecutar al salir del objeto: onmouseout

Ejemplos

  • Click: <button onclick = "funcion1()"> </button>
  • Doble Click: <h4 dblclick = "funcion2()"> </h4>
  • Cursor por encima del Objeto: <p onmouseover = "funcion3()"> </p>
  • Al salir del Objeto: <p onmouseout = "funcion4()"> </p>

Captura por su ID

- Id -

Captura un objeto por su identificador.

  • Cada objeto tiene una 'id' diferente
  • No puede haber dos objetos con una misma 'id'

Las capturas se realizan en el archivo JavaScript asociado al HTML con el que trabajamos. Las capturas se cargan dentro de un apartado específico.

Sintaxis de Carga

window.onload = function() {

// Objetos Capturados

}

Se recomienda declarar las variables antes de la carga de objetos. De este modo se podrán usar en todo el documento.

Ejemplo con ID

// DECLARACIÓN DE VARIABLES

let nombre,

let dato;

// CARGA DE OBJETOS

window.onload = function() {

nombre = document.getElementById("nom"); // Asociada a la id 'nom' del HTML

dato = nombre.value; // Captura el valor de la variable 'id' del HTML

}

// FUNCION A EJECUTAR

function fun_desactivar() {

nombre.hidden = true;

}

Esta función desactivaría el objeto capturado.

Captura por su Clase

- ClassName -

Captura los elementos de una clase concreta, creando un array que contará con todos ellos.

Las capturas se realizan en el archivo JavaScript asociado al HTML con el que trabajamos. Las capturas se cargan dentro de un apartado específico.

Sintaxis de Carga

window.onload = function() {

// Objetos Capturados

}

Se recomienda declarar las variables antes de la carga de objetos. De este modo se podrán usar en todo el documento.

Ejemplo con CLASS

// DECLARACIÓN DE VARIABLES

let imagenes;

imagenes[i].style.border = "1px solid blue";

Esta función pondría a todos los elementos de la clase capturada un border de 1px sólido y de color azul

Ejemplo Elementos Concretos

function fun_estilos2() {

imagenes[2].style.border = "1px solid blue";

}

Esta función pondría al elemento ubicado en la posición '2' (objeto número 3) de la clase capturada un borde sólido de color azul.

Captura querySelector

Captura el primer elemento concreto dentro de un objeto o conjunto de objetos ya capturados.

Las capturas se realizan en el archivo JavaScript asociado al HTML con el que trabajamos. Las capturas se cargan dentro de un apartado específico.

Sintaxis de Carga

window.onload = function() {

// Objetos Capturados

}

Se recomienda declarar las variables antes de la carga de objetos. De este modo se podrán usar en todo el documento.

window.onload = function() {

InnerHTML

Esta función sustituye un objeto por un valor simple.

Ejemplo Sustitución Simple

}

function fun_sustituir() {

nombre.innerHTML = numeroDNI;

}

Esta función sustituiría el objeto 'nombre' por el valor simple 'numeroDNI'.

Sustitución Objetos

En este caso la función extrae el HTML de un objeto para copiarlo encima de otro.

Ejemplo Sustitución Objetos

// DECLARACIÓN DE VARIABLES

let capa;

let capa_imagenes;

// CARGA DE OBJETOS

window.onload = function() {

capa = document.getElementById("capa"); // Asociada a la id 'capa'

capa_imagenes = document.getElementById("imagen"); // Asociada a la id 'imagen'

}

// FUNCION A EJECUTAR

function fun_sustituir2() {

capa.innerHTML = capa_imagenes.innerHTML;

}

En este caso la función innerHTML sustituye un objeto por el contenido de otro objeto. No se copian los estilos, sólo la estructura HTML.

Toggle

Su uso es muy habitual para desplegar menús después de hacer click en un botón. Funciona como un interruptor, activando y desactivando propiedades en CSS en un objeto.

Ejemplo Sintaxis en CSS

/* CARACTERÍSTICAS INICIALES */

#menu-ul {

width: 50%;

margin: 0;

padding: 0;

list-style-image: none;

display: none;

justify-content: center;

}

/* CARACTERÍSTICAS QUE SE ACTIVARÍAN / DESACTIVARÍAN */

#menu-ul.activar {

display: flex;

position: fixed;

top: 0;

right: 0;

}

Ejemplo JS

// DECLARACIÓN DE VARIABLES

let menu;

// CARGA DE OBJETOS

window.onload = function() {

menu = document.getElementById("menu-ul"); // Asociada a la id 'menu-ul'

}

// FUNCION A EJECUTAR

function fun_toggle() {

menu.classList.toggle("activar");

}

En este caso la función 'classList' acompañada de 'toggle' aporta la posibilidad de activar/desactivar características de CSS (a modo interruptor) de la id 'menu'.

addEventListener: Capturados

Los objetos que ejecutarán las funciones deben estar capturados anteriormente.

Sintaxis AddEventListener

// DECLARACIÓN DE VARIABLES

let variable;

// CARGA DE OBJETOS (Se realiza en window.onload)

window.onload = function() {

variable.addEventListener("evento", function() {

código;

});

} // Asociado al objeto que ejecutará la función.

Ejemplo

// DECLARACIÓN DE VARIABLES

let boton;

let menu;

// CARGA DE OBJETOS

window.onload = function() {

boton = document.getElementById("boton-menu"); // Asociada a la id 'boton-menu'

menu = document.getElementById("menu-ul"); // Asociada a la id 'menu-ul'

boton.addEventListener("click", function() {

menu.classList.toggle("activar");

});

} // Asociado al objeto que ejecutará la función.

En este ejemplo, el objeto capturado 'boton-menu' activa una función con un 'click' sin tener que realizar esta configuración en el archivo HTML.

addEventListener: Cualquier Objeto

Los objetos no deben estar capturados con anterioridad. Su uso no es habitual ya que se aplicaría una función desde cualquier objeto de la página.

Sintaxis

// CARGA DE OBJETOS (se realiza en window.onload)

window.onload = function() {

document.addEventListener("evento", function(event) {

event.target.código;

});

}

} // La inclusión de la función 'target' permite detectar cualquier objeto de la página para ejecutar en él la función indicada.

Ejemplo actuación en el propio objeto

// CARGA DE OBJETOS

window.onload = function() {

document.addEventListener("dblclick", function(e) {

e.target.style.border = "1px solid black";

});

}

En este ejemplo, se activa una función que añade un borde sólido de color negro en el objeto sobre el que se ha hecho doble-click.

Ejemplo actuación objeto capturado

// DECLARACIÓN DE VARIABLES

let menu;

// CARGA DE OBJETOS

window.onload = function() {

menu = document.getElementById("menu-ul"); // Asociada a la id 'menu-ul'

document.addEventListener("click", function() {

menu.style.backgroundColor = "lightyellow";

});

}

En este ejemplo, se activa una función cuando se hace click en cualquier objeto, que añade un color de fondo amarillo claro a la id capturada 'menu-ul'.

Sintaxis Teclas Combinadas

Cuando hablamos de sintaxis de tecla compuesta nos referimos a la pulsación de una combinación de teclas que deben accionarse conjuntamente para ejecutar una función. Las teclas más habituales para combinar en una pulsación compuesta son Control, Alt y Shift.

  • Tecla Control: ctrlKey
  • Tecla Alt: altKey
  • Tecla Shift: shiftKey

// CARGA DE OBJETOS (se realiza en window.onload)

window.onload = function() {

document.addEventListener("keydown", function(e) {

if (e.ctrlKey && e.key == "tecla") {

función a ejecutar;

}

});

}

Ejemplo Pulsación Combinada

// CARGA DE OBJETOS

window.onload = function() {

document.addEventListener("keydown", function(e) {

if (e.ctrlKey && e.key == "z") {

fun_reset();

}

});

}

En este ejemplo, se activa la función fun_reset() cuando se pulsan las teclas 'Ctrl' + 'Enter'.

Eventos + Usados en JS

Los eventos más usados para ejecutar funciones desde JavaScript son los siguientes:

  • clik: un click.
  • dblclik: doble-click.
  • mousemove: el cursor pasa por un objeto.
  • mouseout: el cursor sale de un objeto.
  • keydown: el usuario presiona una tecla.

Extracción Tamaño Ventana

Esta función es útil para ejecutar código si el tamaño de la ventana del navegador es uno determinado. También se podría utilizar para saber el dato exacto en cualquier momento.

Ancho de la Ventana

window.innerWidth

Alto de la Ventana

window.innerHeight

Ejemplo

function fun_ventana() {

if (window.innerWidth < 768) {

capa.style.display = "none";

}

else {

capa.style.display = "block";

}

}

En este ejemplo, se ejecuta un código distinto si el tamaño de la ventana es menor o mayor a 768px.

Extracción Tamaño Etiquetas

Para la extracción del tamaño de un objeto o etiqueta se utiliza window.getComputedStyle(), función que devuelve todos los estilos computados del navegador para un elemento.

Gracias al uso de esta función se puede acceder al valor de propiedades tales como width, height, margin, padding,...

Ancho de una Etiqueta

window.getComputedStyle(etiqueta).width

Alto de una Etiqueta

window.getComputedStyle(etiqueta).height

Ejemplo

function fun_etiqueta() {

parrafo.innerHTML = window.getComputedStyle(capa).width;

}

En este ejemplo, se sustituye el objeto con nombre de variable 'parrafo' por el valor del ancho de la variable 'capa'.

ÁREAS VISIBLES

IntersectionObserver() permite ejecutar una función cuando un elemento sale del área visible.

Para su uso adecuado es necesario incorporar la propiedad .isIntersecting, que indica si el elemento está o no visible; y la función .observe() que inicia la observación de un elemento.

Ejemplo

function fun_observar() {

let observar = new IntersectionObserver((entries) => {

entries.forEach(v => {

/* Si la capa NO es visible, mostramos el menú */

if (!v.isIntersecting) {

menu.classList.add("visible"); // Añade la clase CSS "visible" al menú

ancla.style.display = "block"; // Muestra el icono del ancla

}

else {

menu.classList.remove("visible"); // Quita la clase "visible" del menú

ancla.style.display = "none"; // Oculta el icono del ancla

}

});

});

// Empieza a observar el elemento con id o variable "repositorio"

observar.observe(repositorio);

}
