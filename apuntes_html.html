HTML 5
Etiqueta HEAD (Metadatos y Recursos no mostrados)
Es la sección del documento donde se colocan metadatos y recursos que no se muestran directamente en la página, pero que son esenciales para el navegador y los motores de búsqueda.
Etiqueta Significado Ejemplo
<meta> Metadatos <meta charset="utf-8">
<title></title> Título de la Página <title>TÍTULO WEB</title>
<link> Añadir Familia de Fuente de Google Fonts <link href="https://fonts.googleapis.com/css2?family=Anton&display=swap" rel="stylesheet" >
<link> Link de Hoja de Estilos <link rel="stylesheet" type="text/css" href="css/estilo.css">
<style></style> Estilo de la Página <style>p { align: center;}</style>
<script></script> Dinamismo (JavaScript) <script type="text/javascript" src="js/dinamismo.js"></script>
Etiquetas META (Metadatos)
Se ubican en la etiqueta HEAD
Sirven para proporcionar metadatos sobre una página web. Estos datos no se muestran en la página, pero los usan los navegadores, motores de búsqueda, redes sociales y otros servicios.
Etiqueta + Atributo Significado Ejemplo
<meta charset=""> Código de Caracteres <meta charset="utf-8">
<meta name="author" content=""> Autor de la Web <meta name="author" content="Víctor">
<meta name="description" content=""> Descripción de la Web <meta name="description" content="Esta página web...">
<meta name="keywords" content=""> Palabras Clave <meta name="keywords" content="tag1, tag2">
<meta name="robots" content=""> Con Indexeo y Búsqueda <meta name="robots" content="index,follow">
<meta name="robots" content=""> Sin Indexeo ni Búsqueda <meta name="robots" content="noindex,nofollow">
<meta name="viewport" content=""> Ver Responsive en Móviles <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.5, minimum-scale=0.8">
Etiquetas LINK (Vincular Recursos)
Se ubica en la etiqueta HEAD
Se usa en HTML para vincular recursos externos al documento, como hojas de estilo CSS, íconos o fuentes.
Etiqueta + Atributos Significado Ejemplo vinculación archivo (CSS)
<link rel="" type="" href=""> Insertar Hoja de Estilos <link rel="stylesheet" type="text/css" href="css/estilo.css">
Etiqueta STYLE (Estilo)
Se ubica en la etiqueta HEAD
Se usa en HTML para escribir CSS interno, es decir, para incluir reglas de estilo directamente dentro del documento HTML. SE RECOMIENDA EL USO DE HOJAS DE ESTILO.
Atributo Significado Ejemplo añadir propiedades
<style></style> Inserción de estilo en una página <style> p { background-color: blue; } </style>
Etiqueta SCRIPT (Dinamismo)
Se ubica en la etiqueta HEAD
Se usa en HTML para añadir dinamismo a la página a través de JavaScript.
Atributo Significado Ejemplo vinculación archivo (JS)
<script></script> Inserción de dinamismo en una página <script type="text/javascript" src="js/dinamismo.js"></script>
Etiqueta BODY (Cuerpo de la Página)
Define el contenido principal visible de una página web. Es todo lo que aparece en el navegador (texto, imágenes, botones, formularios...)
Atributo Significado Ejemplo
bgcolor Color de Fondo <body bgcolor="purple"></body>
background Imagen de Fondo <body background="img/pantera_rosa.png"></body>
Etiquetas H (Titulos / Encabezados)
Se ubican en la etiqueta BODY
Se usan para definir encabezados o titulos jerárquicos en una página web. Hay seis tamaños disponibles (del 1 al 6), organizados de mayor a menor. Pueden utilizar atributos de identificación como 'id' o 'class'; de estilos como 'style' o de alineación como 'align', aunque se recomienda en estos dos últimos casos utilizar CSS.
Etiqueta Significado Ejemplo
<h1></h1> Titulo (Encabezado) H1 <h1>Titulo H1</h1>
<h2></h2> Titulo (Encabezado) H2 <h2>Titulo H2</h2>
<h3></h3> Titulo (Encabezado) H3 <h3>Titulo H3</h3>
<h4></h4> Titulo (Encabezado) H4 <h4>Titulo H1</h4>
<h5></h5> Titulo (Encabezado) H5 <h5>Titulo H5</h5>
<h6></h6> Titulo (Encabezado) H6 <h6>Titulo H6</h6>
Etiqueta P (Parrafo)
Se ubica en la etiqueta BODY
Se utiliza para definir un parrafo de texto.
Atributo Significado Ejemplo
align="center" Alineación Central <p align="center">Parrafo 1</p>
align="left" Alineación Izquierda (pred.) <p align="left">Parrafo 1</p>
align="right" Alineación Derecha <p align="right">Parrafo 1</p>
align="justify" Alineación Justificada <p align="justify">Parrafo 1</p>
Etiqueta PRE (Texto Preformateado)
Se ubica en la etiqueta BODY
Se usa para mostrar texto preformateado, es decir, respeta los espacios, saltos de linea y tabulaciones tal como están en el código fuente.
<pre> 
El Oso 
											grande 
										es GRIS
</pre>
Etiqueta Marquee (Marquesina en movimiento)
Se ubicaba en la etiqueta BODY
Se usaba para crear texto o contenido que se mueve automáticamente (por ejemplo, de derecha a izquierda), como un efecto de marquesina. No se recomienda en HTML5.
Atributo Significado Ejemplo
bgcolor Fondo de la Marquesina <marquee bgcolor="yellow">Texto</marquee>
behavior="alternate" Movimiento: Rebote <marquee behavior="alternate">Texto</marquee>
behavior="slide" Movimiento: Parada al final <marquee behavior="slide">Texto</marquee>
behavior="scroll" Movimiento: Sale y se pone (pred.) <marquee behavior="scroll">Texto</marquee>
direction="right" Dirección movimiento: Izda. a Dcha. <marquee direction="right">Texto</marquee>
direction="left" Dirección movimiento: Dcha. a Izda. (pred.) <marquee direction="left">Texto</marquee>
direction="up" Dirección movimiento: Abajo a Arriba <marquee direction="up">Texto</marquee>
direction="down" Dirección movimiento: Arriba a Abajo <marquee direction="down">Texto</marquee>
scrolldelay="" Velocidad movimiento (ms) <marquee scrolldelay="2000">Texto</marquee>
Etiqueta BR (Salto de Linea)
Se ubica en la etiqueta BODY
Sirve para insertar un salto de linea dentro de un texto, sin iniciar un nuevo parrafo.
<br>Texto (salto de linea)
Etiqueta HR (Linea de Separacion)
Se ubica en la etiqueta BODY
Se usa para insertar una linea horizontal que visualmente separa contenido dentro de una página.
Atributos Significado Ejemplo
align Alinear parrafo <hr align="center">
width Ancho de la Linea <hr width="80%">
size Tamaño de la Linea <hr size="10px">
color Color de la Linea <hr color="pink">
Etiqueta FONT (Formato del Texto)
Se ubicaba en la etiqueta BODY
Se usaba en versiones antiguas de HTML para cambiar el estilo del texto, como el tipo de letra, color y tamaño. Se recomienda el uso de CSS para cambiar los estilos.
Atributo Significado Ejemplo
color Color de Texto <font color="blue">Texto</font>
size Tamaño de Texto (1 a 7) <font size="7">Texto</font>
face Tipos de Letras (se pueden poner varias por orden) <font face="verdana, arial">Texto</font>
Etiquetas de Resaltado
Se ubican en la etiqueta BODY, concretamente para resaltar fragmentos de texto
Hay varias etiquetas que se usan para resaltar o destacar texto, cada una con un propósito semántico específico.
Etiqueta Significado Ejemplo
<b></b> Negrita <b>Texto</b>
<strong></strong> Negrita (recomendable) <strong>Texto</strong>
<i></i> Cursiva <i>Texto</i>
<u></u> Subrayado <u>Texto</u>
<s></s> Tachado <s>Texto</s>
<sup></sup> Superíndice <sup>Texto</sup>
<sub></sub> Subíndice <sub>Texto</sub>
<mark></mark> Resaltado (color predeterminado: amarillo) <mark>Texto</mark>
Etiqueta UL (Listas Desordenadas)
Se ubica en la etiqueta BODY
Se usa para crear una lista no ordenada, es decir, una lista con viñetas (●, ○,...) en lugar de números.
Atributo Significado Ejemplo
type="circle" Listado Viñeta: Círculo sin relleno <ul type="circle">Texto</ul>
type="disc" Listado Viñeta: Círculo (pred.) <ul type="disc">Texto</ul>
type="square" Listado Viñeta: Cuadrada <ul type="square">Texto</ul>
type="none" Listado Sin Viñeta <ul type="none">Texto</ul>
Etiqueta OL (Listas Ordenadas)
Se ubica en la etiqueta BODY
Se usa para crear una lista ordenada, es decir, una lista con elementos numerados o con otro tipo de marcador ordenado.
Atributo Significado Ejemplo
type="i" Listado: Número Romano (minúscula) <ol type="i">Texto</ol>
type="I" Listado: Número Romano (MAYÚSCULA) <ol type="I">Texto</ol>
type="a" Listado: Letra (minúscula) <ol type="a">Texto</ol>
type="A" Listado: Letra (MAYÚSCULA) <ol type="A">Texto</ol>
start="" Listado: Inicio en Número concreto <ol start="4">Texto</ol>
Listado: Inicio en Número Romano concreto <ol type="i" start="3">Texto</ol>
Listado: Inicio en Letra concreta <ol type="a" start="5">Texto</ol>
Etiqueta TABLE (Tabla)
Se ubica en la etiqueta BODY
Se usa para crear tablas, que organizan datos en filas y columnas.
Atributo Significado Ejemplo
border Borde de Tabla <table border="1"></table>
width Ancho de Tabla (px, %, em) <table width="90%"></table>
align Alineación de la Tabla <table align="center"></table>
height Altura de Tabla (px, %, em) <table height="300px"></table>
bgcolor Color de fondo de la Tabla <table bgcolor="yellow"></table>
background Imagen de fondo de la Tabla <table background="img/pantera_rosa.png"></table>
cellpadding Relleno de la Tabla <table cellpadding="10px"></table>
cellspacing Espacio entre bordes de la Tabla <table cellspacing="0px"></table>
bordercolor Color de borde de la Tabla <table bordercolor="blue"></table>
Etiqueta CAPTION (Título de la Tabla)
Se coloca dentro de la etiqueta TABLE, generalmente justo después de abrir la tabla
Se usa para añadir un título o descripción a una tabla.
<caption>Título de la Tabla</caption>
Etiqueta TR (Filas)
Se coloca dentro de la etiqueta TABLE
Se usa para definir una fila dentro de una tabla. Los atributos que se suelen utilizar son de identificación 'id' o 'class'. El estilo se suele otorgar en TABLE o en las celdas TH o TD.
Fila: <tr></tr>
Etiquetas TH o TD (Celdas)
Se colocan dentro de la etiqueta TR
TH: Representa una celda de encabezado en una tabla. Generalmente el texto aparece en negrita y centrado.
TD: Representa una celda normal de datos en una tabla. El texto de manera predeterminada está alineado a la izquierda.
Atributo Significado Ejemplo
align Alineación Horizontal Celda <td align="center"></td>
valign Alineación Vertical Celda: Arriba <td valign="top"></td>
Alineación Vertical Celda: Abajo <td valign="down"></td>
Alineación Vertical Celda: Centro (pred.) <td valign="middle"></td>
bgcolor Color de fondo de la Celda <td bgcolor="yellow"></td>
background Imagen de fondo de la Celda <td background="img/pantera_rosa.png"></td>
Combinación de Celdas
Se realiza en las etiquetas TH y/o TD mediante los atributos 'colspan' (combinación de celdas de varias columnas) y/o 'rowspan' (combinación de celdas de varias filas).
Atributo Significado Ejemplo
colspan Unir Celdas de Columnas <td colspan="2"></td>
rowspan Unir Celdas de Filas <td rowspan="3"></td>
Etiqueta A (Enlaces)
Se puede ubicar tanto en la etiqueta BODY como en otras etiquetas
Se utiliza para crear hipervínculos (enlaces) que permiten navegar a otras páginas, archivos, secciones del mismo documento, correos electrónicos, o iniciar descargas.
Rutas Relativas (Enlaces): se utilizan para acceder a archivos de la propia web.
Rutas Absolutas (Enlaces): se utilizan para acceder a páginas externas.
Atributo Significado Ejemplo
href Rutas Relativas (Enlaces) <a href="index.html"></a>
Rutas Absolutas (Enlaces) <a href="http://www.google.es/"></a>
Rutas Vacías (Enlaces Ancla) <a href="#"></a>
target="_blank" Apertura en nueva pestaña del navegador (Enlaces) <a href="http://www.diariodeleon.es" target="_blank"></a>
Se recomienda utilizar la etiqueta de enlaces por encima de las etiquetas de parrafo 'p' o elementos de la lista 'li'. Esto permite poder actuar tanto el texto como el posible botón a generar para un menú.
<a href="index.html"><p>Página Principal</p></a>
Enlace de Ancla
Un enlace de ancla en HTML es un tipo de enlace que lleva al usuario a una parte específica de la misma página (o incluso de otra página), en lugar de redirigirlo completamente.
Tipo Ejemplo de lugar al que se quiere acceder Ejemplo Enlace de Ancla
Enlace a una parte específica <a name="palabraclave"></a> <a href="#palabraclave">Título 1</a>
Enlace a una parte específica de otra página <a name="palabraclave"></a> <a href="pag.hmtl#palabraclave">Página</a>
Etiquetas DIV (Capas)
Se puede ubicar tanto en la etiqueta BODY como en otros contenedores
Es un contenedor genérico de bloque en HTML. No aporta significado semántico por sí mismo; simplemente agrupa otros elementos para:
  1. Organizar el marcado, agrupando secciones lógicas de la interfaz.
  2. Aplicar estilos mediante clases o id.
  3. Crear estructuras de maquetación (flexbox, grid, columnas,...).
Atributo Significado Ejemplo
id Identificar un contenedor de forma única <div id="capa1"></div>
class Identificar contenedores a través de clases <div class="amarillos"></div>
style Ancho de un Contenedor <div style="width: 50%;"></div>
Alto de un Contenedor <div style="height: 500px;"></div>
Color de Fondo <div style="background-color: blue;"></div>
Márgenes del Contenedor <div style="margin: 3px;"></div>
Relleno del Contenedor <div style="padding: 5px;"></div>
Espacio entre letras <div style="letter-spacing: 5px;"></div>
Bordes de un Contenedor <div style="border: 1px solid;"></div>
Display de un Contenedor <div style="display: inline-block;"></div>
Tipos de Contenedores
A diferencia de las etiquetas DIV, aportan significado sobre el contenido que contienen, lo que mejora la accesibilidad, el SEO y la organización del código.
Cabecera <header></header>
Menú Superior <nav></nav>
Section <section></section>
Artículo <article></article>
Menú Vertical <aside></aside>
Pie de Página <footer></footer>
Propiedad Display
Regla de estilo para indicar cómo se comporta visualmente un elemento, generalmente se utiliza en los contenedores.
Dato Acción Ejemplo
Block Ocupa todo el ancho (pred.) <div style="display: block;"></div>
Inline Ocupa el espacio del contenido de la etiqueta <div style="display: inline;"></div>
Inline-Block Ocupa el espacio del contenido de la etiqueta, permitiendo modificar tamaño y espaciado <div style="display: inline-block;"></div>
None No se mostaría el contenedor <div style="display: none;"></div>
Etiqueta IMG (Imágenes)
Se puede ubicar tanto en la etiqueta BODY como en otras etiquetas que puedan ejercer de contenedor
Se utiliza para mostrar imágenes en una página web. Hay que tener en cuenta que las imágenes son 'inline', lo que indica que estas se pondrán siempre que quepan en la misma linea.
Atributo Significado Ejemplo
src Ruta de la Imagen <img src="img/logo.png">
style Estilo de la imagen <img src="img/logo.png" style="opacity: 0.8;">
width Ancho de la imagen <img src="img/logo.png" width="300px">
alt Descripción <img src="img/logo.png" alt="Logo de la Empresa">
title Título <img src="img/logo.png" title="Logo">
border Borde de la imagen <img src="img/logo.png" border="1px">
Etiqueta SPAN
Se puede ubicar tanto en la etiqueta BODY como en otras etiquetas
Es un contenedor en linea (inline) sin significado semántico propio. Se usa para aplicar estilos o lógica a una parte específica de texto o contenido en linea, sin alterar el flujo del documento.
<span>La casa del pueblo</span>
Etiqueta VIDEO (Inserción de Vídeos)
Se puede ubicar tanto en la etiqueta BODY como en otras etiquetas contenedoras
Se usa para incrustar videos en una página web. Permite mostrar archivos de video directamente en el navegador, con controles como reproducir, pausar, volumen...
Atributo Significado Ejemplo
src Ruta del Vídeo <video src="video/clip.mkv"></video>
controls Botones de Control <video src="video/clip.mkv" controls></video>
muted Vídeo Silenciado <video src="video/clip.mkv" controls muted></video>
autoplay Inicio Automático (sólo con muted) <video src="video/clip.mkv" controls muted autoplay></video>
loop Repetición en bucle <video src="video/clip.mkv" controls loop></video>
poster Imagen de Portada <video src="video/clip.mkv" controls poster="img/imagenclip.jpg"></video>
width Ancho del Vídeo <video src="video/clip.mkv" controls width="40%"></video>
Inserción de Vídeos Alternativos (SOURCE)
Se ubica dentro de la etiqueta VIDEO
Se usa para proporcionar una o más fuentes alternativas del archivo multimedia. Esto ayuda a asegurar que el navegador pueda reproducir el contenido, incluso si no soporta un tipo de archivo específico.
Existe la posibilidad de incluir una imagen como última alternativa para informar de que el navegador no puede reproducir ninguna alternativa.
<video controls width="40%">
<source src="video/clip.mp4 type="video/mp4">
<source src="video/clip.ogg type="video/ogg">
<img src="img/video_no_soportado.jpg">
</video>

Se puede ubicar tanto en la etiqueta BODY como en otras etiquetas contenedoras
Se usa para insertar archivos de sonido o música en una página web.
Atributo Significado Ejemplo
src Ruta del Audio <audio src="audio/pista.mp3"></audio>

loop Repetición en bucle <audio src="audio/pista.mp3" controls loop></audio>
Inserción de Audios Alternativos (SOURCE)
Se ubica dentro de la etiqueta AUDIO
Se usa para proporcionar una o más fuentes alternativas del archivo de audio. Esto ayuda a asegurar que el navegador pueda reproducir el contenido, incluso si no soporta un tipo de archivo específico.
Existe la posibilidad de incluir una imagen como última alternativa para informar de que el navegador no puede reproducir ninguna alternativa.
<audio controls>
<source src="audio/pista.mp3 type="audio/mpeg">
<source src="audio/pista.ogg type="audio/ogg">
<img src="img/audio_no_soportado.jpg">
</audio>

ATRIBUTOS DE IDENTIFICACIÓN Y CLASE
Atributo Acción Ejemplo
id Identifica una etiqueta
  1. No se puede repetir en otras etiquetas
  2. Son únicos en cada etiqueta de un archivo html
  3. En CSS se representa con: #
 <p id="parrafo1"></p>
class Clasifica etiquetas
  1. Se puede repetir en varias etiquetas
  2. Puede haber varias clases en una etiqueta (las clases se separan con espacios)
  3. En CSS se representa con: .
 <div class="naranjas"></div>
<h1 class="tit menus"></h1>
